Summary: CIF import doesn't take in mind the DB coefficients

Details: Each CIF coordinate on import has to be multiplied by a coefficient (1e-8/DBU)*(a/b) where:
- a and b a coefficients defined by DS command, 
- DBU is the database unit coefficient of the current TDT
- 1e-8 is the CIF integer precision which is constant.
