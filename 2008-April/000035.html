We're happy to announce that Toped 0.9 has been released and is now 
available for download at Berlios.
https://developer.berlios.de/projects/toped/

Fedora package will be available shortly.

The release highlights:
    * Major updates in the user interface including toolbars and new 
panels for the layers and cell hierarchy.
    * Layout libraries with dynamic linking are introduced. Now toped 
can handle multiply layout databases simultaneously.
    * Logic operations are updated with a powerfull resize function 
which works on all layout objects.

Windows package contains also a design example which will be available 
for download from the site shortly.

For further information - visit www/toped.org.uk
